# 飛鏢（Darts）

## 類別

運動機檯類型

## 概述

飛鏢（Darts）是向圓形鏢靶投擲飛鏢、依落點區域計分的遊戲。機檯或電子版以按鈕／觸控決定力道與方向，或模擬投擲；目標常為 **先將分數減至 0**（01 賽制），或單局總分高者勝。本規格涵蓋常見 01 賽制與計分區說明。

## 人數與建議年齡

- 人數：1～多人，通常 2 人對戰或 2 隊。
- 建議年齡：約 8 歲以上。

## 所需器材

- **鏢靶**：標準 20 等分扇形，每區有單倍、雙倍（外圈）、三倍（內圈）；中心為牛眼（50／25 分）。  
- **飛鏢**：實體為三支；機檯為虛擬投擲或按鈕控制。  
- **計分**：每人／每隊有起始分（如 501、301），減至 0 即勝。

## 遊戲目標

**01 賽制**：從約定分數（如 501）開始，每輪三鏢的得分 **從該分數中扣減**；**先恰好在 0 結束** 且最後一鏢須落在雙倍區或牛眼者勝。若爆鏢（扣後小於 0 或剩 1）則該輪不計，分數還原。

## 規則說明

### 準備／設定

- 每人／每隊 501 分（或 301）；決定先攻。一輪 = 投三鏢（實體）或三次投擲（機檯）。

### 回合流程

1. **投擲**：每輪投 3 鏢（或 3 次操作）；每鏢依落點計分。  
2. **計分**：單倍區=該數字、雙倍=×2、三倍=×3；牛眼 50、外圈 25。  
3. **扣減**：該輪 3 鏢總和從當前分數中扣減。  
4. **結束條件**：扣減後 **恰好為 0** 且 **最後一鏢** 落在 **雙倍區或牛眼** → 該方勝。  
5. **爆鏢**：若扣減後 < 0 或 = 1（無法用雙倍／牛眼結束），則該輪 **不計**，分數維持不變。

### 可行動作與限制

- 最後一鏢 **必須** 為雙倍或牛眼（依 01 規則）；其餘鏢無限制。  
- 機檯版可能簡化為：選區域與倍率，或自動判定落點。

## 勝負條件

- **先恰好在 0 結束**（最後一鏢雙倍或牛眼）者勝。  
- **回合賽**：約定局數，先贏 N 局者勝。  
- **高分賽**：單局總分高者勝（無需減至 0）。

## 常見變體或可選規則

- 301、501、701；單/雙人；雙倍入場（第一鏢須雙倍區才開始計分）。  
- Cricket：打掉指定數字與牛眼，先完成且分數不落後者勝。  
- 機檯：自動計分、難度（靶心大小）、練習模式。
